# Healthcare-Analytics-Dashboard-
developed an interactive healthcare dashboard using Microsoft Excel to analyze patient trend, hospital performance, and billing data. Utilized pivot table, pivot charts and slicers to create dynamic KPIs.
# Business Objective
The main objectives of this project are:
- Monitor patient volume and billing trends  
- Identify the most common medical conditions  
- Compare hospital performance  
- Analyze insurance provider contributions  
- Enable interactive filtering for better insights  
# Dashboard KPIs
- Total Patients  
- Total Billing Amount  
- Average Billing per Patient  
# Tools & Techniques Used
- Microsoft Excel  
- Data Cleaning  
- Pivot Tables  
- Pivot Charts  
- Slicers  
- KPI Cards  
- Dashboard Design
# How to Use
1. Download the Excel file  
2. Open in Microsoft Excel  
3. Use slicers to interact with the dashboard  
4. Explore insights across hospitals and conditions
